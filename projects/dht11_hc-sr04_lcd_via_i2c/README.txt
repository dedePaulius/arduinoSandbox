simple project utilizing HC-SR04 Ultrasound distance sensor and DHT11 temperature and humidity sensor.
For result viewing 1602 compatible LCD is used connected via I2C connection.
DHT11 link: http://arduino-er.blogspot.lt/2015/05/arduino-nano-dht11-temperature-humidity.html
LCD via I2C: https://arduino-info.wikispaces.com/LCD-Blue-I2C
Ultrasonic sensor: http://vinduino.blogspot.lt/2013/07/hc-sr04-ultrasonic-sensor.html
For LCD NewLiquidCrystal library is used.
